## 算法 Algorithm
受到[灵茶山艾府的Go语言算法模板库](https://github.com/EndlessCheng/codeforces-go)的启发，决定参考灵茶山艾府的Go模板库建立一个Cpp的算法模板库。由于算法知识点繁杂，将自己学习到的算法、做过的题目分类整理好是有必要的。<br />一个算法模板应当涵盖以下几点：

- 对该算法的基本介绍（核心思想、复杂度等）
- 参考链接或书籍章节（讲的比较好的资料）
- 模板代码（可以包含一些注释、使用说明）
- 模板补充内容（常见题型中的额外代码、建模技巧等）
- 相关题目链接（模板题、经典题、思维转换题等）
