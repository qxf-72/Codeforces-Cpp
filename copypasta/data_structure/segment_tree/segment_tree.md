## 基本介绍

线段树是算法竞赛中常用的用来维护 **区间信息** 的数据结构。线段树可以在 log N 的时间复杂度内实现单点修改、区间修改、区间查询等操作。

线段树能够维护的信息要能够满足 结合律 以及支持快速合并，例如`gcd`、区间求和，求区间最大值，求区间最小值等。
